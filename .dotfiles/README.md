# Dotfiles
My own set of configuration mainly for vim, nvim, tmux and python scripts

``` shell
.config
├── nvim
├── tmux
├── vim
└── zsh
    └── [oh-my-zsh] [1]
.local
├── bin # custom-made scripts
│   ├── md2pdf
│   ├── setapp
│   └── uml2png
├── plugins # adopted scripts / tools
│   ├── [plantuml.jar] [2]
│   └── [saferm.sh] [3]
└── share
    ├── cheatsheets
    └── templates
.zshrc
```
